About Austin PHP Chat

We are a collection of people from various backgrounds. A lot of us attend the Austin PHP Meetup and or the Laravel Austin Meetup groups. Some just write PHP for fun or a living, or use this as a resource for tips, help and networking. We just enjoy talking about code in general.
