#AustinPHP

Join the Austin PHP community on the Freenode IRC network. Use an IRC client from the list below or try the Freenode web chat.

We are a collection of people from various backgrounds. A lot of us attend the Austin PHP Meetup and or the Laravel Austin Meetup groups. Some just write PHP for a fun or a living, or use this as a resource for tips and help, and some just enjoy talking about code in general.
